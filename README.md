\# Achat — Application de Gestion d'Achats



\## Description

Application Spring Boot de gestion d'achats développée dans le cadre du module DevOps/DevSecOps — ESPRIT 2026.



\## Stack technique

\- Java 8 + Spring Boot 2.5.3

\- Maven

\- MySQL (base : achatdb)

\- Swagger UI : http://localhost:8089/SpringMVC/swagger-ui/



\## Lancer l'application

1\. Créer la base de données MySQL : `achatdb`

2\. Configurer `application.properties` avec ton mot de passe MySQL

3\. Lancer : `mvn spring-boot:run`
